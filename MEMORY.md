## Upscaling Pipeline (as of 2026-03-24)
- **x2 (SD2HD)**: nvidia-vfx VideoSuperRes (TensorRT-accelerated) — ~18s/300 frames (~3.3x faster than SPAN)
  - VMAF 91.55 mean, 89.21 min (passes validator threshold of 89)
  - Fallback: SPAN x2 via streaming dual-pipe (~60s/300 frames)
  - Package: `nvidia-vfx` v0.1.0.1 (SDK 1.2.0)
- **x4 (SD24K)**: SPAN x4 via streaming dual-pipe — ~25-35s/300 frames
  - No torch.compile on x4 (not needed, already fast enough)
- **Encoding**: ffmpeg hevc_nvenc pipe for both paths (proven color-correct)
- **PyNvVideoCodec**: Installed (v2.1.0) but NOT used for decode — its NV12→RGB conversion has color matrix mismatch vs ffmpeg (MAE 8.72, causes VMAF drop to 66). Reserved for future GPU-direct encode once ARGB color issue is resolved.
